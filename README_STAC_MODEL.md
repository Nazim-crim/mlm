## Installation

```bash
pip install -U stac-model
```

or install with `Poetry`:

```bash
poetry add stac-model
```
Then you can run

```bash
stac-model --help
```

## Creating an example metadata json

```
stac-model
```

This will make [this example item](./examples/example.json) for an example model.
